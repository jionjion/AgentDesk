AgentDesk

桌面端 + 服务端的智能体工作台。通过自然语言对话与 AI Agent 交互，支持多工具调用、技能管理与定时任务。

GitHub 快速开始

核心功能

💬

智能对话

与 AI Agent 进行自然语言对话，支持 Markdown 渲染与代码高亮显示。

🔧

工具调用

Agent 支持 Function Calling，可灵活扩展工具能力；Agent 本身亦可作为工具被调用。

📋

技能管理

管理和配置 Agent 技能，通过可视化界面进行技能编排。

⏰

定时任务

设置定时任务，让 Agent 在指定时间自动执行任务。

🔍

全文检索

强大的全文检索功能，快速查找历史对话和相关内容。

📦

桌面应用

基于 Electron 构建原生桌面体验，支持系统托盘、快捷操作。

技术栈

🖥️ 前端

框架Vue 3 + TypeScript
桌面端Electron
UI 组件shadcn-vue
样式Tailwind CSS 4
状态管理Pinia
构建工具electron-vite

⚙️ 后端

框架Spring Boot 3.5
语言Java 25
AI 框架AgentScope
数据库PostgreSQL
安全Spring Security + JWT
部署Docker

快速开始

Terminal
# 克隆仓库
git clone https://github.com/jionjion/AgentDesk.git

# 启动后端
cd AgentDesk/backend
mvn spring-boot:run

# 启动前端
cd AgentDesk/frontend
npm install
npm run dev
