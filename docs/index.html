AgentDesk

你的本地 AI 智能体工作台
通过自然语言对话驱动 AI Agent 完成各种任务。支持工具调用、技能编排、定时任务，一切在桌面端完成。
查看源码 快速开始

你能用它做什么

一个桌面端 AI 助手，不只是聊天

💬

智能对话

与 AI Agent 自由对话，回答支持 Markdown 渲染和代码高亮，清晰易读。

工具调用

🔍

全文检索

使用流程

三步开始使用 AgentDesk

部署后端服务

启动 Spring Boot 后端，连接 PostgreSQL 数据库，配置 AI 模型 API Key。

启动桌面应用

运行 Electron 桌面客户端，自动连接后端服务，开箱即用。

对话驱动一切

用自然语言告诉 Agent 你想做什么，它会调用工具帮你完成任务。

快速开始

环境要求：Node.js >= 18 / Java >= 25 / PostgreSQL >= 14 / Maven >= 3.6

后端
# 克隆仓库
git clone https://github.com/jionjion/AgentDesk.git

# 初始化数据库
psql -U postgres -f backend/src/main/resources/database.sql

# 配置环境变量
cd AgentDesk/backend
cp .env.example .env

# 启动后端
mvn spring-boot:run
前端
# 安装依赖并启动
cd AgentDesk/frontend
npm install
npm run dev

# 或构建 Windows 安装包
npm run build:win
