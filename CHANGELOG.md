# Changelog

## AgentDesk v1.1.1

> 稳定性补丁版本！针对后端线程池和标题生成超时问题进行修复，让应用在模型异常时不再卡顿。

---

### 🐛 问题修复

#### ⚙️ 后端稳定性
- **修复标题生成阻塞问题** — 将"自动生成会话标题"的任务从通用线程池隔离到专用线程，避免占用系统公共线程池导致其他异步任务被拖慢
- **修复模型异常时应用卡顿** — 标题生成调用增加 30 秒超时保护，当模型服务响应缓慢或不可用时，不再长时间阻塞后端

---

### ✨ 体验优化

- 首次发起对话时，标题生成更稳健，即便遇到模型超时也能快速降级，不会影响正常对话
- 后端高并发场景下整体响应更平稳

---

### 🔧 工程化改进

- 基于 Electron 33 + Vue 3 + TypeScript 构建桌面应用
- 后端采用 Spring Boot 3 + Java 21 + JPA 架构
- 新增 `RELEASING.md` 发布流程文档，规范每次发版步骤
- 版本号从 v1.1.0 升至 v1.1.1

---

## AgentDesk v1.1.0

> 技能管理与定时任务系统上线！AgentDesk 持续打磨，带来更智能的交互体验和更丰富的定制能力。

---

### 🚀 新功能

#### 🎯 技能管理系统
- **技能配置** — 为 Agent 配置不同技能（翻译、代码审查、写作润色等），让它胜任更多场景
- **闪电按钮** — 点击输入框旁的闪电按钮，自动填入斜杠命令并聚焦输入框
- **斜杠菜单** — 输入 `/` 弹出技能命令菜单，已启用的技能排前面，再次选择可关闭
- **启用状态** — 菜单中清晰显示每个技能的启用/关闭状态

#### ⏰ 定时任务系统
- **调度任务** — 设置定时任务，让 Agent 在指定时间自动工作
- **自动执行** — 无需人工干预，Agent 按计划自动运行
- **状态管理** — 随时查看和管理已创建的任务计划

#### 🔄 对话交互增强
- **模型切换** — 对话中通过下拉框实时切换 AI 模型，不同任务用不同模型
- **图片对话** — 支持在对话中添加图片，视觉模型可理解图片内容
- **消息重新生成** — 对 AI 回复不满意？一键重新生成，获得新回答
- **消息操作栏** — 鼠标悬停助手消息时显示操作栏（复制、重新生成、删除），替代右键菜单

#### 📋 批量操作
- **批量删除会话** — 侧边栏支持全选/多选模式，一次清理多个会话
- **批量管理** — 勾选多个会话后统一操作，管理更高效

#### 📄 导出与下载
- **聊天导出下载** — 后端生成 Markdown 文件直接下载，操作更流畅
- **导出内容完整** — 包含用户消息、AI 回复、思考过程和工具调用记录

#### ⚙️ 设置页增强
- **系统提示词模板** — 模型设置区新增模板下拉选择，快速切换 AI 角色
- **修改密码** — 个人资料区将修改密码从弹窗改为内联表单，体验更自然
- **版本号展示** — 应用设置区显示当前软件版本，方便确认更新

---

### ✨ 体验优化

- 搜索对话框布局调整，限制最大高度，结果区域独立滚动
- 标题栏移除多余菜单按钮，界面更简洁
- 隐私政策页面重新排版，采用左边框+缩进样式，阅读更舒适
- 对话框视觉效果优化，整体更精致
- 统一异常提示风格，报错信息更友好
- 聊天首页菜单选中图标高亮，当前页面一目了然
- SSE 流完成时同步数据库消息 ID，前端与后端数据更一致

---

### 🐛 问题修复

- 修复前端登录/注册页面错误消息提示异常
- 修复设置界面连接跳转问题
- 修复登录相关问题
- 修复关闭前端窗口后后端持续报错的问题

---

### 🔧 工程化改进

- 基于 Electron 33 + Vue 3 + TypeScript 构建桌面应用
- 后端采用 Spring Boot 3 + Java 21 + JPA 架构
- 集成 ESLint 代码规范，统一代码风格
- 集成 GitHub Actions 自动构建 Windows 安装包
- 构建失败自动重试机制（最多 3 次）
- 版本发布说明通过 `releases/` 目录管理，CI 自动生成 CHANGELOG
- 版本号从 v1.0.0 升至 v1.1.0

## AgentDesk v1.0.0

> 首个正式版本发布！AgentDesk 是一款智能体本地工作台，支持与 AI 智能体进行多轮对话、任务规划与工具调用。

---

### 🚀 新功能

#### 🤖 智能对话系统
- **流式对话体验** — AI 回答实时逐字显示，无需等待完整响应
- **思维链展示** — 清晰查看 AI 的思考过程、任务规划和工具调用步骤
- **多轮上下文** — 支持连续多轮对话，AI 能记住之前的交流内容
- **Markdown 渲染** — 回答内容支持 Markdown 格式和代码高亮，阅读更清晰

#### 📋 任务规划系统
- **智能任务分解** — AI 自动将复杂任务拆解为可执行的子任务
- **可视化计划** — 以树状结构展示任务层级和依赖关系
- **进度跟踪** — 实时查看每个子任务的执行状态和完成情况
- **动态调整** — 支持随时修改、添加或删除子任务

#### 🔧 工具调用系统
- **Agent as Tool** — 多个 AI Agent 可以相互协作，分工完成任务
- **工具可视化** — 清晰展示工具调用过程和返回结果
- **扩展能力** — 支持文件分析、写作、编程、数据分析等多种工具
- **实时反馈** — 工具执行状态和结果实时呈现

#### 📎 文件处理系统
- **文件上传** — 支持上传文档、图片等多种格式文件
- **内容解析** — AI 能读取并理解文件内容，与你进行讨论
- **多文件管理** — 一次对话可上传多个文件，方便对比分析

#### 💬 会话管理系统
- **多会话管理** — 创建多个独立话题会话，工作更有条理
- **会话置顶** — 重要会话置顶显示，快速找到常用对话
- **全文检索** — 快速搜索历史对话内容，随时找回所需信息
- **会话删除** — 一键清理不需要的历史会话

#### 📄 聊天导出系统
- **Markdown 导出** — 一键导出聊天记录为 Markdown 格式
- **完整记录** — 导出内容包含用户消息、AI 回复、思考过程和工具调用
- **方便分享** — 导出文件可直接分享或二次编辑

#### 🎨 个性化设置
- **自定义头像** — 设置个人头像，让对话更有个人风格
- **夜间模式** — 深色/浅色主题自由切换，保护眼睛
- **系统托盘** — 关闭窗口后最小化到托盘，后台常驻随时唤回
- **设置中心** — 集中管理应用配置，包括模型参数、界面偏好等

#### 🔐 用户认证
- **邀请码注册** — 支持邀请码机制，控制用户注册
- **本地账号** — 数据保存在本地，隐私安全可控
